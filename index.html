How many operators do we have in JS ?

There are different types of JavaScript operators:

Comparison Operators. =>   ==,===,=>,>=,!=,?
Logical Operators.    =>   &&, ||, !
Conditional Operators. => if   else

Difference between undefined vs not defined vs NaN

            undefined : if variable exists in memory space but hasn't been assigned any value till now, then it would be undefined

            not defined :If the variable name which is being accessed doesn't exist in memory space then it would be not defined

            NaN : NaN stands for Not a Number. It represents a value which is not a valid number.

What is callback hell

        The nested multiple callbacks within a function is called a callback hell.
        example:
        setTimeout(() => {
            console.log(1);
            setTimeout(() => {
                console.log(2);
                setTimeout(() => {
                    console.log(3);
                    setTimeout(() => {
                        console.log(4);
                        setTimeout(() => {
                            console.log(5);
                        }, 1000)
                    }, 1000)
                }, 1000)
            }, 1000)
        }, 1000)
